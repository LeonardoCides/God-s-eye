**Eye of God** is a high-performance OSINT (Open Source Intelligence) tool designed to track a subject's digital footprint across multiple web platforms simultaneously.

Inspired by advanced surveillance systems, this script scans the surface web for specific usernames, delivering a comprehensive report on an individual's online presence.
